igloo

❘ Mobile UI / UX Design ❘ Hotel Booking App ❘

Existing applications that provide hetel reservation services focus only on price comparison. However, the new hotel reservation app, igloo, is designed to allow users to enjoy two functions on one platform, as it includes hotel reservations as well as recommended nearby tourist attractions. Users can communicate with each other by sharing memorable an beautiful places such as tourist attractions, park, restaurants, cafes, and so on.
